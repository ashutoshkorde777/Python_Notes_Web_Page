Python Basics

Python is a popular programming language that is used for a variety of applications, including web development, data analysis, and artificial intelligence.

Installation

To install Python, follow these steps:

  1. Go to the official Python website, which is https://www.python.org/.
  2. Navigate to the "Downloads" section and select the appropriate version of Python for your operating system.
  3. Run the installer and follow the prompts to install Python.

Setup

Once Python is installed, you can set up your development environment. A development environment is a software tool that is used to write, debug, and test code. Here are some popular development environments for Python:

  • IDLE: IDLE is a basic development environment that is included with Python. It is a simple editor that can be used to write and run Python code.
  • PyCharm: PyCharm is a popular integrated development environment (IDE) that is used by many Python developers. It includes a variety of features, including code highlighting, code completion, and debugging tools.
  • Jupyter Notebook: Jupyter Notebook is a web-based development environment that is used for data analysis and visualization. It allows you to create and share documents that contain live code, equations, visualizations, and narrative text.

Running programs

Once you have your development environment set up, you can start running Python programs. A Python program is a file that contains Python code, with a .py extension. Here's how to run a Python program:

  1. Open your development environment and create a new file.
  2. Write your Python code in the file.
  3. Save the file with a .py extension.
  4. Open a terminal or command prompt.
  5. Navigate to the directory where your Python program is located.
  6. Type python filename.py (where filename.py is the name of your Python program) and press enter.

Strings: slicing, concatenation, methods

Strings are a fundamental data type in Python, used to represent text. In Python, strings are defined using single or double quotes, and can be manipulated using a variety of built-in methods. Here is an explanation of some of the most commonly used operations and methods for manipulating strings in Python.

Slicing

Slicing is a method used to extract a portion of a string. The syntax for slicing is string[start:end:step]. The start parameter represents the starting index of the slice, end represents the ending index (exclusive), and step represents the size of the steps taken between indices. 

          
            string = "Hello, World!"
            print(string[7:12]) # Outputs "World"

Concatenation

Concatenation is a method used to join two or more strings together. In Python, you can concatenate strings using the + operator. 

          
            string1 = "Hello"
            string2 = "World"
            string3 = string1 + " " + string2
            print(string3) # Outputs "Hello World"

Methods

Python provides many built-in methods for manipulating strings. Here are some of the most commonly used string methods:

  • upper() and lower(): used to convert the string to all uppercase or lowercase letters, respectively.
  • strip(): used to remove leading and trailing whitespace from a string.
  • replace(): used to replace a specified substring with another substring.
  • split(): used to split a string into a list of substrings.
  • join(): used to join a list of strings into a single string.
    •           
            string = "  Hello, World!  "
            print(string.upper()) # Outputs "  HELLO, WORLD!  "
             print(string.lower()) # Outputs "  hello, world!  "
            print(string.strip()) # Outputs "Hello, World!"

            string = "Hello, World!"
            print(string.replace("Hello", "Hi")) # Outputs "Hi, World!"

            string = "apple,banana,orange"
            fruits = string.split(",")
            print(fruits) # Outputs ['apple', 'banana', 'orange']

            separator = " "
            words = ["Hello", "World!"]
            string = separator.join(words)
            print(string) # Outputs "Hello World!"

    These are just a few examples of the many methods available for manipulating strings in Python. By mastering these fundamental operations and methods, you will be able to work with text data effectively and efficiently in your Python programs.

              empty_list = []

            my_list = [1, 2, 3, 4, 5]
            print(my_list[0]) # Outputs 1

            my_list = [1, 2, 3, 4, 5]
            print(my_list[-1]) # Outputs 5

    To extract a sublist or slice from a list, you can use the slicing operator :. The syntax for slicing a list is my_list[start:end:step], where start is the starting index of the slice (inclusive), end is the ending index of the slice (exclusive), and step is the step size between indices. 

              
           my_list = [1, 2, 3, 4, 5]
           print(my_list[1:3]) # Outputs [2, 3]

    Dictionaries: creating, indexing, methods, common operations

    In Python, a dictionary is an unordered collection of key-value pairs. The key is used to access the value associated with it. Here's an overview of creating and using dictionaries in Python:

    Creating a Dictionary

    To create a dictionary in Python, you can use curly braces {} or the dict() function. Here's an example: 

              
            my_dict = {'apple': 1, 'banana': 2, 'orange': 3}

    In this example, my_dict is a dictionary with three key-value pairs.

    Accessing Dictionary Values

    You can access the value associated with a key by using the key inside square brackets []. Here's an example: 

              
            my_dict = {'apple': 1, 'banana': 2, 'orange': 3}
            print(my_dict['apple'])  # Output: 1

    If the key is not present in the dictionary, it will raise a KeyError.

    Adding and Modifying Dictionary Values

    To add a new key-value pair to a dictionary or modify an existing value, you can use the square brackets []. Here's an example: 

              
            my_dict['pear'] = 4  # Adding a new key-value pair
            my_dict['banana'] = 5  # Modifying an existing value

    Dictionary Methods

    Here are some common methods that can be used with dictionaries in Python:

  • keys(): Returns a list of all the keys in the dictionary.
  • values(): Returns a list of all the values in the dictionary.
  • items(): Returns a list of tuples containing the key-value pairs.

    • Here's an example of how these methods can be used:

              
            print(my_dict.keys())  # Output: dict_keys(['apple', 'banana', 'orange', 'pear'])
            print(my_dict.values())  # Output: dict_values([1, 5, 3, 4])
            print(my_dict.items())  # Output: dict_items([('apple', 1), ('banana', 5), ('orange', 3), ('pear', 4)])

    Dictionary Common Operations

    Here are some common dictionary operations in Python:

  • len(): Returns the number of key-value pairs in the dictionary.
  • in: Returns True if the key is in the dictionary, False otherwise.
  • del: Removes a key-value pair from the dictionary.

    • Here's an example of how these operations can be used:

              
            print(len(my_dict))  # Output: 4
            print('apple' in my_dict)  # Output: True
            del my_dict['pear']
            print(my_dict)  # Output: {'apple': 1, 'banana': 5, 'orange': 3}

    In this example, len() returns the number of key-value pairs in the dictionary, in checks if a key is present in the dictionary, and del removes a key-value pair from the dictionary.

          if condition:
               # code block to be executed if condition is true

    In this syntax, condition is a boolean expression that is evaluated to either True or False. If condition is True, the code block that follows the if statement is executed.

    If-Else Statements

    An if-else statement is used to execute one block of code if a condition is true, and another block of code if the condition is false. Here's the basic syntax of an if-else statement: 

            
          if condition:
              # code block to be executed if condition is true
          else:
              # code block to be executed if condition is false

    In this syntax, condition is a boolean expression that is evaluated to either True or False. If condition is True, the code block that follows the if statement is executed. If condition is False, the code block that follows the else statement is executed.

    If-Elif-Else Statements

    An if-elif-else statement is used to execute different blocks of code depending on multiple conditions. Here's the basic syntax of an if-elif-else statement: 

            
          if condition1:
              # code block to be executed if condition1 is true
          elif condition2:
              # code block to be executed if condition2 is true
          else:
              # code block to be executed if all conditions are false

    In this syntax, condition1 and condition2 are boolean expressions that are evaluated to either True or False. If condition1 is True, the code block that follows the first if statement is executed. If condition1 is False and condition2 is True, the code block that follows the elif statement is executed. If both condition1 and condition2 are False, the code block that follows the else statement is executed.

    2.Loops

    Loops allow you to execute the same code block multiple times. In Python, there are two main types of loops: for loops and while loops.

    for loops

    for loops allow you to iterate over a collection of items. Here's an example: python

            
          fruits = ["apple", "banana", "cherry"]

          for fruit in fruits:
              print(fruit)

    In this example, the for loop iterates over the fruits list and prints each item in the list.

    while loops

    while loops allow you to execute a code block as long as a condition is true. Here's an example: python

            
          i = 0

          while i < 5:
              print(i)
              i += 1

    In this example, the while loop executes the code block as long as i is less than 5. The code block prints the current value of i and then increments i by 1.

    Control Statements

    Control statements allow you to change the flow of a loop or conditional statement. There are two main control statements in Python: break and continue.

    break

    break is used to exit a loop early. Here's an example: 

            
          fruits = ["apple", "banana", "cherry"]

          for fruit in fruits:
              if fruit == "banana":
                  break
              print(fruit)

    In this example, the for loop iterates over the fruits list and prints each item in the list. However, the if statement inside the loop checks if the current item is "banana", and if it is, the break statement is executed, which exits the loop early.

    continue

    continue is used to skip the current iteration of a loop and move on to the next one. Here's an example: 

            
          fruits = ["apple", "banana", "cherry"]

          for fruit in fruits:
              if fruit == "banana":
                  continue
              print(fruit)

    In this example, the for loop iterates over the fruits list and prints each item in the list. However, the if statement inside the loop checks if the current item is "banana", and if it is, the continue statement is executed, which skips the current iteration of the loop and moves on to the next one.

In Python, functions are defined using the def keyword followed by the function name and a set of parentheses. The body of the function is indented and contains the code that is executed when the function is called.

    Here's an example of a simple function that prints a message to the console: 

              
            def print_message():
                print("Hello, World!")

    In this example, the function name is print_message. When the function is called, it will print the message "Hello, World!" to the console.

    Calling a Function

    To call a function in Python, you simply need to write the name of the function followed by a set of parentheses. If the function takes parameters, you can pass them inside the parentheses. Here's an example: 

              
            def print_name(name):
                print("Hello, " + name + "!")

            print_name("John")

    In this example, the function print_name takes one parameter name and prints a message to the console using that parameter. When the function is called with the argument "John", it will print the message "Hello, John!" to the console.

    Lambda Function

    A lambda function in Python is a small, anonymous function that can have any number of arguments, but can only have one expression. Lambda functions are also known as "anonymous functions" because they don't have a name.

    Lambda functions are defined using the lambda keyword, followed by a list of arguments and a single expression. The syntax for a lambda function is as follows: 

              
            lambda arguments: expression

    Here's an example of a simple lambda function that adds two numbers: 

              
            add_numbers = lambda x, y: x + y
            result = add_numbers(2, 3)
            print(result)

    In this example, the lambda function takes two arguments x and y, and adds them together using the expression x + y. The result of the lambda function is assigned to the variable add_numbers, and then the lambda function is called with the arguments 2 and 3. The result of the lambda function call (5) is then printed to the console.

    Lambda functions can be used in a variety of ways in Python. One common use case is to pass a lambda function as an argument to another function. For example, the built-in map() function in Python can be used to apply a function to every element in a list. Here's an example: 

              
            numbers = [1, 2, 3, 4, 5]
            squares = list(map(lambda x: x ** 2, numbers))
            print(squares)

    In this example, the lambda function lambda x: x ** 2 is passed as the first argument to the map() function. The lambda function takes one argument x and returns the square of that argument using the expression x ** 2. The map() function applies this lambda function to every element in the numbers list, and returns a new list of the squares of each number. The result of the map() function call is then converted to a list using the list() function, and printed to the console.

    Lambda functions are a powerful tool in Python, and they can be used to write concise and readable code. However, it's important to use them judiciously, and to keep in mind that they can sometimes make your code harder to understand if they're overused or used inappropriately.

    File Handling

    File handling is an important part of many programming languages, including Python. Python provides built-in functions to read and write files, as well as libraries to handle more complex file formats. In this explanation, we'll cover the basics of file handling in Python.

    Opening a file:

    Before you can read or write a file in Python, you need to open it. The built-in open() function is used to open a file. The open() function takes two arguments: the filename, and the mode in which the file is to be opened. The mode can be "r" for reading, "w" for writing, "a" for appending, or "x" for exclusive creation.

    Here's an example of how to use the open() function to open a file for reading: 

              
            file = open("example.txt", "r")

    In this example, the open() function is used to open the file "example.txt" in read mode. The file variable is now a file object that can be used to read the contents of the file.

    Reading from a file:

    To read the contents of a file in Python, you can use the file object's read() method. The read() method reads the entire contents of the file as a string.

    Here's an example of how to read the contents of a file: 

              
            file = open("example.txt", "r")
            contents = file.read()
            print(contents)

    In this example, the read() method is used to read the entire contents of the file. The contents of the file are then printed to the console using the print() function.

    You can also use the readline() method to read one line at a time from a file. The readline() method reads one line at a time, and returns an empty string when it reaches the end of the file.

    Here's an example of how to read one line at a time from a file: 

              
            file = open("example.txt", "r")
            line = file.readline()
            while line:
                print(line)
                line = file.readline()

    In this example, the readline() method is used to read one line at a time from the file. The while loop is used to read and print each line in the file until the end of the file is reached.

    Writing to a file:

    To write to a file in Python, you can use the file object's write() method. The write() method writes a string to the file.

    Here's an example of how to write to a file: 

              
            file = open("example.txt", "w")
            file.write("Hello, world!")
            file.close()

    In this example, the write() method is used to write the string "Hello, world!" to the file. The close() method is then used to close the file.

    Appending to a file:

    To append to a file in Python, you can use the file object's write() method with mode a. The write() method writes a string to the file, and appends it to the end of the file.

    Here's an example of how to append to a file: 

              
            file = open("example.txt", "a")
            file.write("Hello, again!")
            file.close()

    In this example, the write() method is used with mode a to append the string "Hello, again!" to the end of the file.

    Closing a file:

    When you're done working with a file in Python, it's important to close it using the file object's close() method. Closing the file releases any system resources that were being used by the file.

    math

    The math module provides various mathematical functions and constants. Here are some examples of how to use the math module in Python: 

              
            import math

            # Square root of a number
            x = math.sqrt(25)
            print(x)
            
            # Trigonometric functions
            y = math.sin(0.5)
            z = math.cos(0.5)
            print(y, z)
            
            # Constants
            pi = math.pi
            e = math.e
            print(pi, e)

    In this example, we import the math module and use its sqrt() function to compute the square root of 25. We also use the sin() and cos() functions to compute the sine and cosine of 0.5 radians. Finally, we use the pi and e constants provided by the math module.

    Random

    The random module provides functions for generating random numbers and sequences. Here are some examples of how to use the random module in Python: 

              
            import random

            # Random integer between 1 and 10
            x = random.randint(1, 10)
            print(x)
            
            # Random choice from a list
            my_list = [1, 2, 3, 4, 5]
            y = random.choice(my_list)
            print(y)
            
            # Shuffle a list
            random.shuffle(my_list)
            print(my_list)

    In this example, we import the random module and use its randint() function to generate a random integer between 1 and 10. We also use the choice() function to randomly select an element from a list, and the shuffle() function to randomly shuffle the elements of a list.

    time

    The time module provides functions for working with dates and times. Here are some examples of how to use the time module in Python: 

              
            import time

            # Current time in seconds since the epoch
            x = time.time()
            print(x)

            # Current local time as a string
            y = time.ctime()
            print(y)

            # Pause the program for 5 seconds
            time.sleep(5)
            print("Done sleeping")

    In this example, we import the time module and use its time() function to get the current time in seconds since the epoch (January 1, 1970). We also use the ctime() function to get the current local time as a string. Finally, we use the sleep() function to pause the program for 5 seconds.

    NumPy Library

    NumPy is a Python library for numerical computing with arrays and matrices. It provides tools for working with large, multi-dimensional arrays and matrices, along with a variety of mathematical operations that can be performed on them. NumPy is a fundamental library for scientific computing with Python, and is widely used in data analysis, machine learning, and other fields.

    Here are some of the key features and functions of the NumPy library:

  • Multi-dimensional arrays: NumPy provides a powerful and flexible multi-dimensional array object called ndarray. This object can be used to represent vectors, matrices, and other types of arrays, and provides efficient operations for manipulating them.
  • Broadcasting: NumPy supports broadcasting, which allows you to perform operations on arrays of different shapes and sizes. This makes it easy to write code that can work with arrays of varying dimensions.
  • Mathematical functions: NumPy provides a wide range of mathematical functions that can be performed on arrays, including trigonometric, logarithmic, and exponential functions.
  • Linear algebra operations: NumPy provides a set of functions for performing linear algebra operations on arrays, including matrix multiplication, inversion, and eigenvalue computation.
  • Fourier transforms: NumPy includes functions for computing fast Fourier transforms (FFTs), which are useful in signal processing and other applications.

    • To use the NumPy library in a Python program, you first need to import it using the following statement: 

              
            import numpy as np

    This will load the NumPy library into your program and allow you to use its functions and classes. Here are a few examples of how to use NumPy in Python: 

              
            import numpy as np

            # Create a one-dimensional array
            a = np.array([1, 2, 3, 4, 5])
            
            # Create a two-dimensional array
            b = np.array([[1, 2], [3, 4]])
            
            # Perform element-wise addition on two arrays
            c = a + b
            
            # Compute the dot product of two arrays
            d = np.dot(a, b)
            
            # Compute the sine of an array
            e = np.sin(a)
            
            # Compute the eigenvalues of a matrix
            f = np.linalg.eigvals(b)

    In the above example, we create a one-dimensional array a, a two-dimensional array b, and perform element-wise addition and dot product on them. We also compute the sine of the a array, and the eigenvalues of the b matrix using NumPy functions. Note that NumPy functions are usually faster than using pure Python for similar computations.

    Broadcasting rules

    Broadcasting is a mechanism in NumPy that allows arrays with different shapes to be used in arithmetic operations. The broadcasting rules in NumPy define how arrays with different shapes are treated during arithmetic operations. Here are the key rules:

  • If the arrays have different ranks (number of dimensions), the lower-ranked array is padded with ones on its leading (left) side.
  • If the arrays have the same number of dimensions, but different shapes, the array with shape equal to 1 in any dimension is stretched to match the corresponding shape of the other array.
  • If the arrays have different shapes, and none of the shapes are equal to 1, a ValueError is raised, indicating that the arrays are not compatible.

    • Broadcasting allows for efficient computation by eliminating the need to create intermediate arrays. It also simplifies the syntax of operations involving arrays of different shapes. For example, you can add a scalar value to an array or add two arrays of different shapes using the same syntax as you would use for two arrays of the same shape.

    Overall, broadcasting is a powerful feature of NumPy that makes it easier to work with arrays of different shapes and sizes, and allows for efficient and concise computation.

    Matplotlib Library

    Matplotlib is a popular Python library used for creating visualizations, such as line plots, scatter plots, bar plots, histograms, and more. It provides a wide range of customization options and supports a variety of output formats, including interactive displays and static images.

    Here is a brief overview of how to use Matplotlib in Python:

    Before using Matplotlib, you need to import the library using the following command:

              
            import matplotlib.pyplot as plt

    To create a plot, you first need to create some data to visualize. Then, you can use the plot() function to create a line plot, for example:

              
            x = [1, 2, 3, 4]
            y = [2, 4, 6, 8]
            plt.plot(x, y)

    Customizing the plot:

    Once you have created a plot, you can customize it by adding a title, labels for the axes, legends, and more. For example:

              
            x = [1, 2, 3, 4]
            y = [2, 4, 6, 8]
            plt.plot(x, y)

    Customizing the plot:

    Once you have created a plot, you can customize it by adding a title, labels for the axes, legends, and more. For example:

              
            plt.title("My Plot")
            plt.xlabel("X-axis label")
            plt.ylabel("Y-axis label")
            plt.legend(["Line 1"])

    Displaying the plot: After creating and customizing the plot, you can display it using the show() function: 

              
            plt.show()
